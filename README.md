# Cycloid Explorer

브라우저에서 하이포사이클로이드와 에피사이클로이드를 시각화하는 작은 정적 웹 프로젝트입니다.

큰 원의 반지름 `a`, 작은 원의 반지름 `b`, 그리고 작은 원의 각속도를 직접 입력해 보면서 내부 또는 외부를 구르는 작은 원과 점 `P`의 궤적을 관찰할 수 있습니다.

## Features

- `a`, `b`, 속도를 텍스트 입력으로 직접 조절
- `3/2`, `sqrt(2)`, `pi`, `2*pi` 같은 식 입력 지원
- 하이포사이클로이드, 에피사이클로이드, 동시 보기 모드 지원
- 큰 원 안쪽 또는 바깥쪽을 따라 반시계방향으로 구르는 작은 원 애니메이션
- 정지 또는 일시정지 상태에서 큰 원과 작은 원을 각각 숨기거나 함께 숨기기 가능
- 그리드 포함 또는 제외 PNG 저장 지원
- `a / b`가 유리수로 판정되면 한 주기 후 자동 정지
- 잘못된 입력이 들어오면 이전의 유효한 값을 유지

## Files

- [index.html](./index.html): 페이지 구조
- [styles.css](./styles.css): 레이아웃과 스타일
- [script.js](./script.js): 입력 파싱, 애니메이션, 궤적 계산

## Run

별도 빌드 없이 바로 실행할 수 있습니다.

1. 이 폴더에서 [index.html](./index.html)을 브라우저로 엽니다.

또는 간단한 로컬 서버를 띄워서 열어도 됩니다.

```bash
python3 -m http.server
```

그다음 브라우저에서 `http://localhost:8000`으로 접속하면 됩니다.

## Input Rules

- `a > 0`
- `b > 0`
- 하이포사이클로이드가 선택된 경우 항상 `b < a`
- 속도는 `0`보다 커야 함
- 허용 예시: `4`, `1`, `1/3`, `sqrt(2)`, `pi/8`, `2*pi`

## Default Values

- `a = 4`
- `b = 1`
- `speed = 0.02`

## Notes

- 점 `P`의 시작점은 `(a, 0)`입니다.
- 속도 값은 한 프레임당 증가하는 각도(`rad/frame`)입니다.
- `a / b`가 무리수인 경우에는 곡선이 닫히지 않을 수 있으므로 자동 정지하지 않습니다.
- 오른쪽 위 `저장` 메뉴에서 현재 화면을 PNG로 내려받을 수 있습니다.
